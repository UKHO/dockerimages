# terraform-win

## Purpose
Windows Server Core image that provides Terraform for Windows-based CI/CD jobs.

## Installed dependencies
| Dependency | Version | Notes |
|---|---|---|
| Terraform CLI | 1.10.5 | Downloaded from HashiCorp releases |
| Windows Server Core base image | ltsc2022-amd64 | From `mcr.microsoft.com/windows/servercore:ltsc2022-amd64` |
